to run a php symfony project:
open terminal, go in the project folder, run symfony server:start, click on the link it give you in the green square,
then add the route of the homepage (or the special page you want to go to) example : http://127.0.0.1:8000/lucky/number
remember to use tue URL written in the routes.yaml

npm run watch to enable css modification to take effect immlediately

CTRL+C in the used terminal close the symfony server
